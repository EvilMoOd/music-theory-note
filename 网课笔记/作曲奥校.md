# 作曲奥校



## 新和声学



### 绪论

1. 和声学不能独立存在，依赖于某个时期的文化背景，音乐也不只是音乐，跨学科融合已渐渐兴起
2. 音乐发展的五个阶段，原始音乐、传统音乐：严格的真善美，共性；现代音乐：打破共性，极致的个性，物极必反导致美学变丑学，创新大于情绪；当代音乐：实用主义，与商业结合的各种影游或各种流行作品，色域论，意趣大于创新；后当代音乐：大数据和AI带来的扁平化、垂直化
3. 五个阶段的划分依据分别为：记谱法，思想革命和工业革命，信息革命，大数据革命
4. 不同阶段音乐因不同地域文化发展而定，而且都存在于如今受不同人喜爱，2、4阶段是如今主流
5. 和声理论具有局限性，无法全然解释345阶段的音乐
6. 三种行为模式：理论型、情感型、科技型
7. 三种理论：他型理论（规定主义）-前人的各种乐理规则，写出符合大众审美的音乐，适用于2、4阶段的音乐创作；我型理论（描述主义）-以自我为中心，不受条条框框的限制，适用于1、3阶段的音乐创作。你型理论（建构主义）-特定场合、特定人群来定制音乐
8. 同一种事物基于不同的背景下会有不同的结果，比如251并不一定就是完全好听的。学习作曲理论要以宏观为前提再去具体到微观的技术，**以某种技法能达到特定效果** 的思路学习，实用、实践为主
9. 作曲大方向上分为学术和专业，对应第一三人称和第二人称



### 流行和声套路基础

1. 标记一元论
2. 同一套路进行使用不同属性
3. 和声节奏，主副哥以2的倍数小节划分，偶尔出现切分节奏，前奏、间奏、尾奏会有更复杂的和声节奏
4. 和声以2度5度进行为主
5. 和声分析不能片面的局部分析或只想套路，要联系歌词、旋律、表演等多个元素，以作品整体看待。比如绘词法、单曲循环播放
6. 音乐加入音效、台词等元素
7. 和声不必以主和弦开始或结束



### 流行循环套路总结

1. 分层和弦套路（要注意某些和声标记可以整体解释的分层优先用整体标记），顺阶、非顺阶的组合
2. 民族音乐
3. 调式音乐、短循环



### 唱腔

1. 布鲁斯唱腔，357音适当降低些音分，不严格按平均律，另外可以加点swing
2. 润腔就是加花、不按谱即兴发挥的流行唱法，流行音乐不润腔会显得呆板不自然所以必须润腔。相对的古典音乐就禁止润腔，为了体现庄重
3. 音色变化是润腔的延伸效果



### 线条化和声

1. 低音线条、旋律线条、内声部线条，原位、转位，平行和声
2. 多声部线条。某些复杂的和弦标记，可能是线条化的结果
3. 二部框架向下构造的经过和声
4. 线条化时可能会产生对斜，古典中避免对斜（小二度等），但在爵士流行中经常用，和声或是旋律唱腔（蓝音），在使用时要谨慎处理，好听就能用



### 和声定型进行

1. 离调251（套娃进行），有时候会伴随半音接近，属于和声润腔
2. 转置进行，循环的和声进行，可以从不同起点开始
3. 消化效应：因为和弦是三度叠置的，所以三度向上进行时共同音声部会被消化，缺乏流动性。可以用变化音、转位、和弦外音的方式避免消化效应。消化效应并不是必须避免的，电声乐器泛音强的特性可以忽略消化效应



### 音乐科技

1. 提到踏板、daw这些对音乐发展的蝴蝶效应
2. 音色叠加、音高叠加、琶音器等当代音乐技术
3. 分析和声不能只靠和声理论，还要考虑科技、人文、社会等因素
4. 拼贴音乐：加入各种其他素材
5. 偶成和弦：和声线条偶然形成的和弦

4. 1级大九和弦可以看成属和弦向下延伸两个音

## 转调法

### 单声部转调法

1. 当前的旋律声部持续到转调后，用同音自然地进入到新调中
2. 可以音阶级进到新调
3. 总结就是单音或是过渡的手段进入到新调的旋律或是和声声部



### 平移转调法

1. 句中上推
2. 句后上推
3. 加料上推：加料可以是前乐句延伸、静默，先用新调的和声让听众习惯
4. 旋律引导的平移转调
5. 平移离调



### 共同音转调法

1. 句末型（等号在前型）
2. 句后型（等号在后型）
3. 句中型（等号在中型）
4. 等音转调



## 旋律复调

### 旋律写作

1. 主题鲜明，不要写重复音高旋律（除了rap）还有和声型旋律
2. 调性明确，级进、跳进、装饰，重复、模进加深旋律印象
3. 乐句呼吸，注意演唱的气口



### 传统复调对位五大点

1. 一对一，三度六度对位
2. 一对二，经过辅助音之类的技巧，主要音拴住次要音
3. 一对四，注意避免四度、分解和弦、非结构点五八度、自由音、同向
4. 一对切分
5. 一对混合
